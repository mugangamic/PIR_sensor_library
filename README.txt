This is an library for Leaflabs's Maple. Library supports series of low cost PIR sensors. 

This is a simple library for PIR sensing.
Written by Oskari Rauta.

To download: click the DOWNLOADS button in the top right corner, rename the uncompressed folder PIR_sensor_library. Check that the folder contains PIR.cpp and PIR.h. Place the library folder your <maplesketchfolder>/libraries/ folder. You may need to create the libraries subfolder if its your first library. Restart the IDE. Connect PIR's output to
PIN15 and run example. Check the serial output.
